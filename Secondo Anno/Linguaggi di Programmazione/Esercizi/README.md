
## Alcune tip sui comandi che userai

### L'ambiente di lavoro
Personalmente non ho avuto intenzione di installarmi tutto l'ambiente Prolog per spargermi roba in giro
per il Mac, ma io sono una fighetta, si sa. Pertanto ho utilizzato direttamente [questo sito]{https://swish.swi-prolog.org/}

help. Si spiega da sè, ti dà le indicazioni e l'help di come funziona Prolog

working_directory(CWD, '/Users/*utente*/*' in cui CWD è Current Working Directory

listing(*predicato*). Ti mostra tutte le regole che parlano di *predicato

edit(*nomeProg*). Serve per editare un programma con emacs

edit(*nomeProg*). SE il file non esiste ti suggerisce di creare il nuovo file direttamente da console

make. Ricompila il programma dopo l'edit
